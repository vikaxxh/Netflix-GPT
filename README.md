# 🎬 IMDb Clone (React + Vite)

A modern **IMDb Clone** built using **React** and **Vite**, designed to browse movies, view details, and explore trending content with a clean UI and fast performance.

---

## 🚀 Features

* 🔍 Search for movies
* 🎞️ View movie details (title, rating, overview, etc.)
* 📈 Trending / Popular movies section
* ⚡ Fast performance using Vite
* 🎨 Responsive UI

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/imdb-clone.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd IMDB-CLONE
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run Development Server

```bash
npm run dev
```

### 5️⃣ Open in Browser

```
http://localhost:5173/
```
